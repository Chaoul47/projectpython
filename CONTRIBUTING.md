# Contributing

Thanks for helping improve Sonic Cipher.

## Quick Start
1. Fork the repo and create a feature branch.
2. Install dependencies: `pip install -r requirements.txt`.
3. Make your changes.
4. Run a quick compile check: `python -m py_compile app.py Sonic_Cipher/*.py`.
5. Open a pull request with a clear description and screenshots if UI changes.

## Style
- Keep code readable and small.
- Avoid changing behavior unless required.
- Prefer clear error messages.

## Reporting Issues
Use the issue templates in GitHub and include:
- Steps to reproduce
- Expected vs actual behavior
- Platform and Python version
